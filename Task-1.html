Basics Of Web Development

// HTML Section //

HTML (Structure)

HTML (HyperText Markup Language) defines the structure of a webpage. It organizes content using tags.

Basic HTML Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>Welcome to My Page</h1>
    <p>This is a paragraph.</p>
    <img src="logo.png" alt="Sample Image">
    <a href="#">Click Here</a>
  </body>
</html>

Key Elements:
<h1>...</h1> - Used for headings
<p>...</p> - Used for paragraphs
<img src="..." alt="..."> - Used to display images
<a href="...">...</a> - Used for links
<button>...</button> - Used for clickable buttons

// CSS Section //

CSS (Styling)

CSS (Cascading Style Sheets) is used to design the appearance of a webpage by adding colors, fonts, spacing, and layouts.

Basic CSS Structure:

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}

h1 {
  color: navy;
  text-align: center;
}

p {
  font-size: 18px;
  color: #333;
}

Key CSS Properties:
color - Sets text color
font-size - Sets size of text
background-color - Sets background color
text-align - Aligns text
margin, padding - Adjust spacing

// JavaScript Section //

JavaScript (Interactivity)

JavaScript adds interactivity to a webpage. It can respond to user actions like clicks, input, or movement.

Basic JavaScript Structure:

<script>
function showMessage() {
  alert("Hello! You clicked the button!");
}
</script>
