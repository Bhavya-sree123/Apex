Basics Of Web Development

Understand the structure and syntax of HTML, CSS, and JavaScript

HTML (Structure)

HTML defines the structure of a webpage. It organizes content using tags.

Basic HTML Structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My Webpage</title>
  </head>
  <body>
    <h1>Welcome</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Key Elements:
<h1>...</h1> - Headings
<p>...</p> - Paragraphs
<img> - Images
<a> - Links
<button> - Buttons

CSS (Styling)

CSS is used to design the look of a webpage by adding colors, fonts, and layouts.

Basic CSS:

body {
  background-color: lightblue;
  font-family: Arial;
}

h1 {
  color: navy;
  text-align: center;
}

Key CSS Properties:
color - Text color
font-size - Text size
background-color - Background color
text-align - Text alignment
margin, padding - Spacing

JavaScript (Interactivity)

JavaScript adds interactivity to a webpage and responds to user actions.

Basic JavaScript:

<script>
function showMessage() {
  alert("Hello! You clicked the button!");
}
</script>
